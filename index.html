Getting Started

Welcome to the Climate AI Prediction System. This project uses deep learning to forecast temperature severity and extreme weather events.

  1. Clone the repository to your local machine.
  2. Ensure you have Python 3.9+ installed.
  3. Run the initialization script to set up your environment.

Download Packages

To run the climate models, you need the following dependencies:

pip install pandas numpy scikit-learn tensorflow matplotlib

These libraries handle the LSTM layers and data preprocessing required for severity analysis.

User Manual

Detailed instructions on how to use the prediction interface:

  • Data Input: Upload CSV files containing historical weather data.
  • Severity Check: The system automatically flags "Extreme" if the prediction exceeds 2 standard deviations.
  • Export: Results can be exported as a PDF report.

Machine Learning Models

Our project utilizes a Hybrid Model approach:

  • LSTM (Long Short-Term Memory): Used for predicting 10-day temperature trends.
  • Random Forest: Categorizes the severity of predicted heatwaves or droughts.
